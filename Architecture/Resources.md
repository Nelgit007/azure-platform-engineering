# Azure Resources Reference

| Category | Resource | Description |
|----------|------------|-------------|
| **General** | Resource Group | Logical container for Azure resources. |
| **General** | Subscription | Billing and management boundary for Azure resources. |
| **General** | Management Group | Organizes multiple subscriptions for governance. |
| **General** | Tags | Metadata used for resource organization and cost tracking. |
| **General** | Azure Policy | Enforces organizational standards and compliance. |
| **General** | Azure Blueprints | Defines repeatable environments with policies and templates. |
| **General** | Azure Monitor | Collects, analyzes, and acts on telemetry data. |
| **General** | Log Analytics Workspace | Central repository for logs and monitoring data. |
| **General** | Microsoft Entra ID | Identity and access management service (formerly Azure AD). |
| **General** | Key Vault | Secure storage for secrets, certificates, and keys. |
| **Compute** | Virtual Machine (VM) | On-demand scalable virtual servers. |
| **Compute** | Virtual Machine Scale Set (VMSS) | Automatically scales VM instances. |
| **Compute** | Availability Set | Provides fault and update domain separation for VMs. |
| **Compute** | Availability Zone | Physically separate datacenters within a region. |
| **Compute** | Azure Kubernetes Service (AKS) | Managed Kubernetes container orchestration service. |
| **Compute** | Container Instances (ACI) | Run containers without managing servers. |
| **Compute** | App Service | Fully managed platform for web applications and APIs. |
| **Compute** | Azure Functions | Event-driven serverless compute service. |
| **Compute** | Batch | Runs large-scale parallel and high-performance computing jobs. |
| **Compute** | Dedicated Host | Physical servers dedicated to a single customer. |
| **Storage** | Storage Account | Provides access to Azure storage services. |
| **Storage** | Blob Storage | Object storage for unstructured data. |
| **Storage** | Azure Files | Fully managed file shares accessible via SMB/NFS. |
| **Storage** | Azure Disks | Persistent block storage for Azure VMs. |
| **Storage** | Queue Storage | Message storage for asynchronous processing. |
| **Storage** | Table Storage | NoSQL key-value data store. |
| **Storage** | Data Lake Storage Gen2 | Scalable analytics storage for big data workloads. |
| **Storage** | Archive Storage | Low-cost storage for infrequently accessed data. |
| **Storage** | Backup Vault | Centralized backup management for Azure resources. |
| **Storage** | Recovery Services Vault | Stores backup and disaster recovery data. |
| **Networking** | Virtual Network (VNet) | Private network for Azure resources. |
| **Networking** | Subnet | Segments a VNet into smaller networks. |
| **Networking** | Network Security Group (NSG) | Controls inbound and outbound network traffic. |
| **Networking** | Application Security Group (ASG) | Groups VMs for NSG rule management. |
| **Networking** | Public IP Address | Publicly routable IP for Azure resources. |
| **Networking** | Load Balancer | Distributes traffic across multiple resources. |
| **Networking** | Application Gateway | Layer 7 load balancer with WAF capabilities. |
| **Networking** | Azure Firewall | Managed cloud-native firewall service. |
| **Networking** | VPN Gateway | Secure site-to-site and point-to-site connectivity. |
| **Networking** | ExpressRoute | Dedicated private connection to Azure. |
| **Networking** | Bastion | Secure RDP/SSH access without exposing public IPs. |
| **Networking** | NAT Gateway | Provides outbound internet connectivity. |
| **Networking** | Private Endpoint | Private access to Azure services over a VNet. |
| **Networking** | Private DNS Zone | DNS resolution for private resources. |
| **Networking** | Traffic Manager | DNS-based global traffic distribution. |
| **Networking** | Front Door | Global application delivery and acceleration service. |
| **Networking** | Route Table | Controls traffic routing within VNets. |
| **Networking** | Peering | Connects two VNets privately. |
